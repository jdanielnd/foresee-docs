Para você se familiarizar com o aplicativo, vamos percorrer um pequeno exemplo do começo ao fim.

Para isso, disponibilizamos um arquivo de dados do qual você vai precisar mais à frente. Portanto, clique no link abaixo com o botão direito, clique em "Salvar link como...", e salve-o em alguma pasta de seu computador.

Vendas por Região.csv

Etapa 1: cadastre uma senha para sua conta.

Você recebeu um e-mail com o assunto "Bem vindo ao Foresee!", que contém um link para que você possa acessar sua conta e criar uma senha. Clique neste link e siga as instruções.

Etapa 2: acesse a sua página de projetos.

Para acessar sua página de projetos, basta acessar https://app.fores.ee. Se você não estiver logado, o Foresee vai solicitar seu e-mail e senha.

Etapa 3: crie um novo projeto.

Criar um novo projeto é muito simples, basta clicar em "Novo projeto", um botão azul situado no lado superior direito da página de projetos, e preencher algumas informações:

  • Nome do projeto: dê um nome para o seu projeto, por exemplo, "Vendas por Região".
  • Arquivo de dados: é o arquivo com os dados históricos das séries que deseja projetar. Selecione o arquivo que você salvou anteriormente em seu computador.
  • Intervalo de predição: é o nível de confiança da projeção. Vamos falar sobre isso mais tarde. Inicialmente, deixe como está (80%).

Em seguida, clique em "Criar projeto". O projeto será salvo e você será redirecionado para a tela desse projeto. As projeções ainda não foram calculadas. Repare que o status do projeto é "Processando", e existe uma barra de progresso indicando o percentual das projeções que já foi calculado.

Aguarde até que os cálculos terminem e a página ser atualizada, exibindo uma lista com as projeções contidas no projeto. Neste caso: Região I, Região II e Região III.

Etapa 4: acesse as projeções.

Para acessar cada uma das projeções, basta clicar no nome da projeção que deseja visualizar. Clique sobre "Região I".

Você será levado à página dessa projeção. Nessa página é possível visualizar o gráfico da projeção e também uma tabela, contendo os valores projetados. Mais à frente vamos entender o que significa cada uma dessas colunas.

Dica: Use as setas para a direita e esquerda do seu teclado para navegar entre as projeões.

Etapa 5: exporte as projeções.

Se quiser salvar esses resultados no seu computador, ou mesmo compartilhá-los com outras pessoas da empresa, basta exportá-los em CSV ou PDF.

Para fazer isso, retorne à página do projeto clicando sobre o nome do projeto (Vendas Mensais) na barra de navegação localizada acima do menu de navegação, como apresentado abaixo:

Para exportar todas as projeções contidas nesse projeto, clique no link para o formato desejado - por exemplo, clique em PDF - ao lado de "Projeções" no quadro de informações do projeto.

Para exportar uma projeção individual, localize-a na tabela e clique no formato desejado apresentado na coluna Download. Por exemplo, clique em CSV para exportar o arquivo CSV com os resultados da Região I.

Parabéns! Você percorreu todas as etapas para criar e explorar um projeto no Foresee.
