Para você se familiarizar com o aplicativo, vamos percorrer um pequeno exemplo do começo ao fim.

Criamos um arquivo de dados de exemplo, e você vai precisar dele mais à frente. Portanto, clique no link abaixo com o botão direito, clique em "Salvar link como...", e salve-o em alguma pasta de seu computador.

Vendas mensais 2013.csv

Etapa 1: cadastre uma senha para sua conta.

Você recebeu um e-mail com o assunto "Bem vindo ao Foresee!", que contém um link para que você possa acessar sua conta e criar uma senha. Clique neste link e siga as instruções.

Etapa 2: acesse a sua página de projetos.

Para acessar sua página de projetos, basta acessar https://alpha.fores.ee. Se você não estiver logado, o Foresee vai solicitar seu e-mail e senha.

Etapa 3: crie um novo projeto.

Criar um novo projeto é muito simples, basta clicar em "Novo projeto", um botão azul situado no lado superior direito da página de projetos, e preencher algumas informações:

  • Nome do projeto: dê um nome para o seu projeto. Por exemplo, "Vendas Mensais".
  • Arquivo de dados: é o arquivo com os dados históricos das séries que deseja projetar. Selecione o arquivo que você salvou anteriormente.
  • Intervalo de predição: é o nível de confiança da projeção. Vamos falar sobre isso mais tarde. Inicialmente, deixe como está (80%).

Em seguida, clique em "Criar projeto". O projeto será salvo e você será redirecionado para a tela dele. As projeções ainda não foram calculadas. Repare que o status do projeto é "Processando", e existe uma barra de progresso indicando o percentual das projeções que já foi calculado.

Aguarde até que os cálculos terminem, que a página atualizará, exibindo uma lista com as projeções contidas no projeto. Neste caso: Região A, Região B, Região C e Região D.

Etapa 4: acesse as projeções.

Para acessar cada uma das projeções, basta clicar em seu nome. Por exemplo, experimente clicar sobre "Região A".

Você será levado à página da projeção. Nesta página é possível visualizar o gráfico da projeção, e uma tabela, indicando os valores da projeção. Mais a frente vamos entender o que significa cada uma dessas colunas.

Etapa 5: exporte as projeções.

Se quiser salvar em seu computador alguns desses resultados, basta exportá-los para CSV ou PDF.

Para fazer isso, retorne para a página de projeto clicando no nome do projeto (Vendas Mensais) na barra de navegação localizada acima do menu de navegação, como apresentado abaixo:

Home / Projetos / Vendas Mensais / Região A

Para exportar todas as projeções do projeto, clique no link para o formato desejado ao lado de "Projeções" na tabela de informações do projeto.

Para exportar uma projeção individual, localize-a na tabela e clique no formato desejado apresentado na coluna Download.

Parabéns! Você percorreu todas as etapas para criar e explorar um projeto no Foresee.
