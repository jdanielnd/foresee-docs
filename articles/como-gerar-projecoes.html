As projeções geradas a partir de um mesmo arquivo de dados ficam armazenadas dentro de um projeto, o que facilita o acesso posterior aos resultados. Você pode manter em sua conta quantos projetos quiser!

Para criar um projeto clique no botão Novo Projeto.

Você será automaticamente direcionado para uma nova página (ver imagem abaixo), onde deverá inserir as informações necessárias para que o novo projeto seja criado.

Para isso:

  1. atribua um nome ao projeto.
  2. selecione, no seu computador, o arquivo de dados com o histórico das séries que deseja projetar.
  3. escolha o intervalo de predição.
  4. clique no botão “Criar Projeto”.

Pronto! Em poucos instantes você será automaticamente direcionado para a página do projeto que acabou de criar (ver imagem abaixo).

Clicando sobre o nome de qualquer uma das séries contidas neste projeto (Região I, por exemplo), você acessa o gráfico e a tabela com as projeções relativas àquela série (ver imagem abaixo).
