The forecasts you generate with Foresee are filed into projects, which you can access at any time, either from your desktop, tablet or smartphone.

Beyond that, Foresee allows you to export the results of your forecasts into CSV files - which can be opened in Microsoft Excel Â® - or, if you prefer, into PDF reports which contain plots and tables of your forecasts.

To access the forecasts stored within a project, click on the name of the particular project.

You will be directed to the project page you selected.

  1. Clicking on CSV, to the right of "Data File" will allow you to access the file whose data were used to generate the forecasts in that particular project.
  2. Clicking CSV to the right of "Forecasts", you'll access the file CSV with all forecasts in this project.
  3. Clicking on PDF, to the right of "Forecasts", you'll access a report in PDF format with plots and tables covering all the forecasts in this project.
  4. Clicking on CSV next to a particular series, you'll access the file CSV with forecasts from that series.
  5. Clicking on PDF next to a particular series, you'll access a report in PDF format with the plot and table relating to that series.
  6. Hovering the mouse over the name of a series, you will notice that a trash can icon will appear to the right side. Clicking on this icon allows you delete the selected series. This same procedure applies to projects that you have created. On the project page, hovering your mouse over the name of a project and clicking on the trash will let you delete that particular project.
