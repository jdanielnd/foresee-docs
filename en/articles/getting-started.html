To familiarize yourself with the application, let's go through all the steps of a small sample project.

We have provided a data file, which you will need later. Right click the link below, click 'Save link as ...' and save it to any folder on your computer.

Monthly sales 2013.csv

Step 1: Register a password for your account.

You've received an email with the subject "Welcome to Foresee", which contains a link that enables you to access your account and create a password. Click this link and follow the instructions.

Step 2: Go to your Project page.

To access your Projects page, simply go to https://app.fores.ee. If you're not logged in, Foresee will ask for your email and password.

Step 3: Create a New Project.

Creating a new project is very simple - just click on the blue 'New project' button on the upper right of the project page and fill in the requested information:

  • Project Name: give a name for your project (for example, "Monthly Sales").
  • Data file: This is the file containing the historical data series you want to project. For this example, select the file that you saved previously to your computer.
  • Prediction interval: This is the confidence level of the forecast. We’ll get into this in more detail later. For now, leave it as is (80%).

Next, click 'Create project'. The project will be saved, and you will be redirected to the window for this project. The forecasts have not yet been computed. Note that the status of the project is 'Processing', and there is a progress bar indicating the percentage of the forecasts that have been calculated.

Wait until Foresee finishes the calculations. The page will refresh, displaying a list of the forecasts contained in the project. In this case: Product 1, Product 2 and Product 3.

Step 4: Go to the forecasts.

To access each of the forecasts, just click on the name of the forecast you would like to view. For example, click on "Product 1".

You will be taken to the page for the "Product 1” forecast. On this page you can view the forecast plot and also a table containing the forecasted values. Later we’ll explain what each of these columns represents.

Tip: Use the left and right arrows from your keyboard to navigate through the forecasts.

Step 5: Export the forecasts.

If you want to save these outputs on your computer or share them with others in the company, you can export them as CSV or PDF files.

To do this, return to the project page by clicking on the project name (Monthly Sales) in the navigation bar, which is located above the navigation menu, as shown below:

To export all the forecasts included in the current project, click the link for the desired format. For example, you might click 'PDF' nexto to "Forecasts" in the project information area.

To export a single forecast, locate it in the table and click the desired format displayed in the 'Download' column. For example, click 'CSV' to export the CSV file containing the forecasts for the Product 1.

Congratulations! You have completed all the steps necessary to create and explore a project in Foresee.
