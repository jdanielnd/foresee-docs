The forecasts generated by the same data file are stored inside a project, which makes further access to the results easier. You can keep as many projects as you want in your account!

To create a project, click on New project.

You will be automatically redirected to a new page (see image below), where you'll need to provide the necessary information in order to create a new project.

Therefore:

  1. give your project a name.
  2. select, in your computer, the data file with the series history that you would like to project.
  3. chose the prediction range.
  4. click on “Create Project”.

You're ready! In a few moments you will be automatically redirected to the project page, which you just created (see image below).

By clicking on any series name that is on this project (Region I, for instance), you'll get to the graphic and table with the forecasts related to such series (see image below).
