Foresee is an extremely simple and easy to use forecasting app.

First Steps

As you can see in the First steps section, in order to create forecasts you only need to load a data file with the history of the series you wish to forecast.

Formatting a data file

To generate forecasts, the only thing you need to be careful about is that the data file be correctly formatted. Therefore, we strongly recommend that you read carefully the How to format a data file section.

Reading forecasts

Although you don't need to be an "expert" in statistics to use the app (the concept behind Foresee is exactly that!), it's important to know how to read a forecast. The How to read forecasts section presents the essential concepts you need to know to use forecasts in your business.

Support

The help section includes the main questions that you might have regarding the use of the app.

If you have any questions or suggestions that aren't answered in the help section, please send us an e-mail at hello@fores.ee. We'll be happy to help you!
