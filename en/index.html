Foresee is an extremely simple and easy to use app.

First Steps

As you can see in the First steps section, in order to create forecasts, you only need to load a data file with the history of the series that you wish to project.

Formatting a data file

The only thing you need to be careful about is that the file be correctly formatted. Therefore, we recommend that you read carefully the How to format a data file section.

Understanding the forecasts

Although you don't need to be an "expert" in statistics to use the app - the concept behind Foresee is exactly that! - it is important that you know how to interpret a forecast. The How to understand forecasts section summarizes in a very didactic way all you need to know how to use safely Foresee forecasts.

Support

We seek to include at the Help section of Foresee the main questions that you may possibly have regarding the use of the app.

Moreover, if you have any questions or suggestions, send us an e-mail at hello@fores.ee. We are available to help you!
